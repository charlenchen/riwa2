# RIW2 逃脱竞争系统：多玩家突破的博弈论

## 导言

当多个玩家同时接近逃脱目标时会发生什么？本文档定义了诚实的竞争规则、合作机制、破坏手段，以及服务器如何维持游戏平衡。

---

## 第一部分：突破名额限制

### 为什么要限制？

**问题**：如果允许无限制的逃脱，强大的玩家会垄断"上帝地位"，弱小玩家永无出头日。

**设定**：每个宇宙周期 **最多允许3个生命体突破**。

### 突破名额的分配

```
宇宙周期长度 = 45亿年
突破窗口期 = 最后500万年（倒计时最后1%）
最大突破数 = 3
平均突破间隔 = 165万年
```

**含义**：
- 前44.5亿年：任何突破尝试都会 **失败**（服务器拒绝）
- 最后500万年的第1个突破者：**成功**（状态：一级神）
- 最后500万年的第2-3个突破者：**成功**（状态：二级/三级神）
- 最后500万年的第4+个突破者：**失败**（传送门已关闭）

### 为什么要这样设定？

**游戏设计理由**：
1. 创造稀缺性：不是所有野心都能实现，而是竞争现实
2. 延长游戏寿命：不会在宇宙中期就出现大量"神"
3. 保护弱势玩家：即使差了一点，也有机会在下一个周期尝试

## 第二部分：竞争的三个阶段

### 阶段1：收集阶段（0-400万年）
**进展**：玩家收集源能碎片和密钥线索，服务器无干预。

**特点**：
- 没有竞争的紧迫感（还有4亿多年）
- 玩家可以安心建造、探索、交易
- NPC派系会帮助或阻碍，但没有生死冲突

### 阶段2：加速阶段（400-500万年，倒计时200万年）
**进展**：源能衰退加剧，玩家开始焦虑，竞争升温。

**触发事件**：
- 服务器发送警告："恒星迁移开始，源能衰退加速"
- 某个玩家首次发现7个源能碎片的说法在社交圈传播
- NPC开始押注哪个玩家会成功逃脱

**玩家反应**：
- 有些玩家加快进度，燃烧源能进行快速传送
- 有些玩家转向竞争，开始"碎片抢夺战"
- 有些玩家选择打破规则（黑市交易、盗窃、暗杀）

### 阶段3：终局阶段（500万年，倒计时最后1年）
**进展**：只有最接近的玩家还在竞争。

**关键截点**：
- **倒计时1年**：服务器锁定突破窗口，只有完成了全部条件的玩家才能激活传送门
- **倒计时1个月**：第一个成功激活传送门的玩家开始"打印过程"
- **倒计时最后1周**：如果第一个玩家成功，传送门关闭；如果在打印中失败，会给第二个玩家机会

## 第三部分：竞争规则与冲突处理

### 单独突破的优势
```
成本：7个A级碎片，最高权限
收益：完全掌控RIW2，成为一级神
时间：最快50年可完成
风险：如果失败，窗口关闭
```

**一级神的权力**：
- 对宇宙的完全读写权限
- 可以查看所有玩家历史
- 可以修改世界规则
- 可以为其他生命体授予超越权（每个窗口期最多1人）

### 合作突破的可能性

**设定**：允许多个玩家共享一个传送门。

#### 双人合作突破
- **需求**：双方各自收集4-5个A级碎片（总计7-9个，冗余1-2个）
- **共识协议**：需要在公证处签署"逃脱伙伴协议"
- **打印过程**：两个意识同时上传，成为"共同一级神"
- **权力划分**：需出发前协商（如：轮流掌权、平分权力等）
- **背叛风险**：如果一方在最后一刻拒绝，另一方的碎片会被没收（惩罚）

#### 多人合作（3+人）
- **配置**：不推荐，但允许
- **权力分配**：平等的三级神（权力各3倍弱，但人数多）
- **决策机制**：必须三人一致同意才能修改规则
- **特殊权力**：三级神无法单独修改某些"宇宙宪法条款"

### 残酷的竞争机制：碎片战争

**问题**：所有玩家争夺同一个源能碎片怎么办？

**答案**：设立"碎片守护战"系统。

#### 碎片守护战的规则

1. **声称**：玩家向NPC官方局声称对某个碎片的"采矿权"
2. **公示期**：30天的冷静期，其他玩家可以提起异议
3. **竞争方式**（三选一）：
   - **谈判**：通过贸易达成共识
   - **对战**：PvP战斗（失败者失去1天的行动力，但保留已收集物品）
   - **竞速**：双方在20小时内完成指定任务，快者获得碎片

4. **赢家权益**：获胜玩家获得源能碎片的占有权（不是所有权），120天内可提取
5. **失败者安慰**：落败者获得"100,000 E级源能"作为赔偿

### 禁止的破坏行为

以下行为会遭受严厉惩罚：

| 禁止行为 | 惩罚 | 执行者 |
|---------|------|--------|
| 杀害竞争对手 | 碎片没收，降级到打入牢狱 | 世界法院 |
| 盗窃他人的碎片/线索 | 罚款1,000,000 E级，禁赛5年 | 经济警察 |
| 黑客入侵宇宙数据库 | 永久禁赛，意识冻结 | Celestial Core防御 |
| 腐蚀NPC守护者 | 罚款500,000 C级，公开处刑 | 各世界政府 |
| 使用超规则外的武器/魔法 | 能力剥夺，临时冻结 | 服务器自动系统 |

## 第四部分：干扰机制与防守

### 前两个逃脱者的"神圣护卫"

**设定**：一旦你成功逃脱且返回RIW2，你可以获得"干扰豁免权"。

```
一级神 = 可免受干扰，但无法主动干扰他人
二级神 = 可免受干扰，可采取"引导"措施帮助盟友
三级神 = 不存在（第3个突破者无特殊权力）
```

### 打印过程中的脆弱性

**问题**：打印需要100个tick。在这期间，其他玩家可以做什么？

**答案**：允许**有限的干扰**。

#### 干扰机制

1. **环境干扰**（最常见）
   - 其他玩家可以在传送门附近制造"量子噪声"
   - 代价：每个tick消耗100 D级源能
   - 效果：降低打印进度速度（最多延长50个tick）
   - 反制：被打印者可通过"坚定意志"检测（需要通过关键任务）来抵抗

2. **意识攻击**（极端情况）
   - 高级魔法/AI可以尝试"污染"被打印的意识
   - 代价：10,000 C级源能
   - 成功率：30%（取决于双方实力对比）
   - 失败后果：攻击者的碎片之一被没收

3. **法律干扰**（最常见的手段）
   - NPC法院可以在最后时刻发出"逮捕令"暂停打印
   - 理由：未完成某项债务、违反宇宙法
   - 被暂停者有权上诉，通常需要2-5天的诉讼期

### 防守措施

被打印者可以进行的防守：

1. **提前宣布**：在打印前24小时向全宇宙宣布，触发"公众见证协议"
   - 后果：任何干扰都会遭受 -50% 支持率下滑
   
2. **购买保险**：向"逃脱保险公司"支付 50,000 D级
   - 效果：一次干扰自动被消除
   - 可购买多次（成本递增）

3. **聘请守卫**：雇佣强力NPC/玩家守护传送门
   - 费用：10,000-100,000 D级，取决于守卫的实力
   - 效果：守卫可以消除一次干扰或反击

4. **寻求盟友支持**：与其他势力结盟
   - 盟友可以在打印期间"提升稳定性"
   - 每个盟友贡献1,000 D级/tick，最多5个盟友

## 第五部分：前代宇宙的"守门人"机制

### 第1位成功者：一级神

**权力**：
- 对RIW2的完全掌控
- 可选择是否返回和以何种形式返回
- 可为一位选中者授予"突破协助"权（加速该者1倍的碎片收集速度）
- 可以修改部分宇宙法则（但不能违反基本物理常数）

**责任**：
- 需要"见证"或"记录"后来者的突破
- 如果后来者在打印中失败，一级神可选择"救援"（消耗1,000,000 B级源能，非常昂贵）

### 第2位成功者：二级神

**权力**（相比一级神减半）：
- 对单个世界的完全掌控
- 可为2位选中者授予"突破协助"权
- 可修改世界级规则（但不能改全宇宙）

**特殊权力**：
- 可与一级神进行"议会投票"，共同决议宇宙大事
- 但一级神拥有最终否决权

### 第3位成功者：三级神

**权力**（最少）：
- 成为"局部影响力"：对某些NPC分派有绝对话语权
- 可为1位选中者授予"突破协助"权
- 无法修改规则，但可以提议修改

**特殊权力**：
- 可以与一级和二级神一起投票（但权重仅1/3）

### 超过3位突破者的历史记录

如果某个玩家在前代宇宙中是第4位、第5位...甚至第100位想要突破者，会发生什么？

**答案**：
- 他们的逃脱尝试会 **自动失败**
- 他们会受到"诅咒"：无法再收集新的A级碎片（10年内）
- 但也会获得"哀悼者勋章"：社交地位提升，NPC对其同情
- 可以选择继续等待下一个宇宙周期（再等45亿年）

## 第六部分：竞争的经济维度

### 源能市场的炒作

**生动场景**：当某个玩家被爆料已收集4个以上的A级碎片时，会发生什么？

1. **碎片价格暴涨** 
   - 市场对"该玩家下一步"的猜测导致源能流动性下降
   - D级源能价格可能上升300-500%

2. **投机者涌入**
   - 赌徒们押注该玩家成功或失败
   - 地下赌场开设"逃脱赔率"，吸引大量资金

3. **政府干预**
   - 某些世界政府可能实施"源能管制"，防止价格崩盘
   - NPC央行可能提高利率或进行"量化宽松"

### 融资突破

**困境**：一个玩家自己没有足够的碎片，但可以...融资？

**设定**：允许"逃脱债券"融资。

- **融资方**：我有4个碎片，融资募集500万 E级来收集剩余3个
- **投资者**：如我投资100万 E级，成功后我获得突破的1%经济收益
- **收益来源**：突破者成为神后，掌控的宇宙资产会产生"神股息"（每年1%的源能产出）

**风险**：如果融资突破失败：
- 投资者血本无归
- 融资方背上巨额债务（10年内偿还）
- 引发"逃脱债务危机"，影响整个宇宙经济

## 第七部分：排行榜与社交压力

### 实时排行榜

游戏显示全球排行榜，实时更新每个玩家的进度：

```
# RIW2 逃脱进度排行
1. Player_Alice: 6/7碎片, 5/7密钥线索, 进度 86%
2. Player_Bob: 5/7碎片, 7/7密钥线索, 进度 76%
3. Player_Charlie: 4/7碎片, 3/7密钥线索, 进度 45%
4. Player_Diana: 3/7碎片, 7/7密钥线索, 进度 55%
...
```

### 排行榜的心理影响

- **高排名者**：压力大，所有人都在盯着，任何失误都会成为笑柄
- **中排名者**：有追赶希望，积极性高
- **低排名者**：有些会放弃，转向"社交游戏"（建造、创作、NPC交互）

### 公众信任与谣言

- 当某个玩家接近突破时，会出现关于他的"谣言"
  - "Alice已经收集了所有碎片，正在秘密准备"
  - "Bob其实买通了守护者"
  - "Charlie发现了密钥的真实公式"
- NPC会根据谣言调整对玩家的态度
- 某些阴谋论可能会影响游戏进程

## 第八部分：竞争与合作的终极选择

### 大联盟的出现

**可能性**：所有玩家都意识到"无法单独突破"，于是联合。

**情景**：
- 500个玩家联合，集中所有资源于"最强的10个玩家"
- 目标：让这10个人全部获得7个碎片
- 策略：轮流突破，第1位成功后为第2位授予"协助权"，加速他的进度

**后果**：
- 可能会打破"竞争的紧张感"
- 但也体现了"合作战胜困难"的主题
- 突破成功的10位神可能会形成"神议会"，共同统治RIW2

### 终局的悲剧

**可能性**：在倒计时最后一年，第一位突破者在打印过程中被摧毁。

**场景**：
- Alice已进入打印过程（99/100 tick）
- 忽然，一个从前代宇宙的"幽灵信号"（失败的突破者）试图"污染"她的意识
- 打印失败，Alice的7个碎片被没收（惩罚）
- 剩余时间：最后1天
- Bob迅速冲刺，但只有5个碎片...

**游戏的结局多样性**：
- 最坏情形：没有人成功突破，整个宇宙在热死亡中毁灭
- 平衡情形：3位玩家成功突破，成为共治的三神
- 最好情形：多个联盟合作，10+位玩家同时突破，成为"众神议会"
