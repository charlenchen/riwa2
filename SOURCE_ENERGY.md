# RIW2 源能系统：完整经济学

## 导言

源能是RIW2宇宙的基础货币和物理资源。理解源能系统是理解整个游戏经济的关键。本文档澄清了源能的生成、流通、消耗、再生和最终归宿。

---

## 第一部分：源能的基础物理

### 源能的本质

**定义**：源能是从Celestial Core（服务器）驱动的恒星核聚变反应中获得的量子纠缠能量。

**物理性质**：
- 来源：恒星的聚变反应
- 传输：通过"源能场"均匀分布到所有虚拟世界
- 特性：既可被消耗，也可被部分回收
- 度量：以"A级当量"为统一单位

### 恒星与源能产率

```
恒星能量产率 → 转化为源能 → 分配到RIW2系统
```

**当前驻扎恒星**：太阳（Sol-001）
- 聚变功率：3.828 × 10²⁶ 瓦
- RIW2分配比例：0.001%（服务器运行需求）
- 年产源能：≈ 1,000,000 A级当量
- 理论寿命：45亿年

**关键设定**：恒星的寿命 = 宇宙周期的最大时长。当恒星寿命即将耗尽时，Celestial Core必须迁移到下一颗恒星，触发"大迁移事件"（见下文）。

## 第二部分：源能的等级系统

### 五级分类

| 等级 | 名称 | 相对价值 | 再生周期 | 用途 |
|------|------|--------|--------|------|
| **E** | 普通 | 1 | 2-4周 | 日常生活、简单操作 |
| **D** | 稀有 | 100 | 1-3个月 | 跨界旅行、属性强化 |
| **C** | 珍稀 | 10,000 | 半年 | 历史改写、世界修补 |
| **B** | 传奇 | 1,000,000 | 1-2年 | 维度通讯、生命转移 |
| **A** | 神圣 | 100,000,000 | 永不再生 | 终极传送门的唯一燃料 |

### 等级说明

**E级（普通源能）**
- 日常消耗：NPC存活维持、简单法术/技能
- 再生机制：通过"生命活动代谢"自动产生
  - 每个活跃NPC/玩家每周产生 1-5 E级源能
  - 生物学意义：智慧生命体维持意识的副产品
- 可交易：可自由交易
- 数量：最丰富，占总量的60%

**D级（稀有源能）**
- 用途：跨界旅行（100 D级 = 一次长距离传送）、能力强化
- 再生机制：通过"知识积累"和"技能训练"产生
  - 完成重要任务：+50 D级
  - 学习新技能：+10 D级
  - 创新发明：+100-1000 D级
- 可交易：可交易，但税费15%
- 数量：占总量的20%

**C级（珍稀源能）**
- 用途：历史修改（改变过去事件）、世界修补（修复世界规则漏洞）
- 再生机制：通过"文明进步"和"文化突破"产生
  - 建立新的文明制度：+500 C级
  - 创作艺术杰作：+100 C级
  - 宗教或哲学创新：+200 C级
- 可交易：交易需要政府/NPC派系审批
- 数量：占总量的15%

**B级（传奇源能）**
- 用途：维度间通讯、生命转移、世界融合
- 再生机制：通过"超越性事件"产生
  - 发现宇宙秘密：+1000 B级
  - 建立跨界文明联盟：+5000 B级
  - 首次发现某个关键真理：+10000 B级
- 可交易：极少，仅由服务器和最高权力者掌握
- 数量：占总量的4%

**A级（神圣源能）**
- 用途：**唯一用途**：激活终极传送门
- 再生机制：**永不再生**
  - 初始分配：宇宙创建时放入7个A级碎片
  - 消耗后：永久消失，对应宇宙衰退
- 不可交易：不可交易，不可合成，只能收集
- 数量：占总量的1%（但影响最大）

### Ω级（已消耗）
在前代宇宙中，可能存在过更高等级的源能（Ω级）。这些能量已被耗尽，成为"宇宙衰退"的证据。前代宇宙生命体的幽灵信号可能就是这场能量消耗战的遗留。

## 第三部分：源能流通与消耗

### 源能的出口（消耗点）

```
总源能产出 = 所有出口消耗之总和
```

#### 1. 生命维持消耗
- **对象**：所有活跃实体（NPC、玩家、AI）
- **消耗速率**：
  - 每个玩家/NPC：每周 5-20 E级（取决于活跃度）
  - 每个被动NPC：每月 1-2 E级
  - 每个世界系统维护：每自然周期 1,000 E级
- **特点**：无法避免；停止消耗 = 实体冻结/死亡

#### 2. 跨界操作消耗
- **传送（单程）**：100 D级 （约 10,000 E级当量）
- **信息传递（维度间通讯）**：10 B级/次
- **物品转移**：500 D级/件
- **时间同步**：1,000 C级/次（协调不同世界的时间流速）

#### 3. 历史改写消耗
- **改变过去1小时**：100 C级
- **改变过去1天**：500 C级
- **改变过去1年**：10,000 C级
- **改变过去1个世纪**：100,000 C级（需要政府批准）
- **改变大事件**：1,000,000+ C级（极少发生，需全球投票）

**警告**：每次历史改写都会创建"时间支流"，增加宇宙的熵，加速热死亡。

#### 4. 世界维护与修复
- **修复单个规则漏洞**：500-5,000 C级
- **大型世界升级**：100,000 C级
- **世界崩溃恢复**：500,000 C级（从备份恢复）
- **跨界融合**：1,000,000 B级（两个世界合并）

#### 5. NPC特殊能力消耗
- **施放强力法术**：10-1000 D级/次
- **机器人执行任务**：1-100 E级/小时
- **AI思考复杂问题**：100-10,000 D级/次

### 源能的回收机制

**并非所有消耗的源能都永久丧失。** 部分可以被回收：

#### 回收率表
| 源能类型 | 回收率 | 回收途径 |
|---------|------|--------|
| E级 | 80% | 过期生命体解散、自然衰减 |
| D级 | 50% | 历史改写的"时间余波"、失败的跨界操作 |
| C级 | 10% | 被否决的改写方案、修复完成后的剩余 |
| B级 | 5% | 失败的维度实验 |
| A级 | 0% | 永不回收 |

**回收过程**：消耗的源能进入"宇宙熵池"，缓慢被Celestial Core重新捕获，周期为3-6个月。

## 第四部分：源能与宇宙衰退

### 永久性消耗与热死亡

**关键设定**：虽然源能可部分回收，但每个回收周期都会损失能量（熵增）。

**衰退公式**：
```
宇宙衰退 = 总消耗量 × (1 - 平均回收率) × 时间系数
```

**实际含义**：
- 第一个100万年：衰退 ~2%
- 第二个100万年：衰退 ~5%（更多生命体 = 更多消耗）
- 第三个100万年：衰退 ~10%
- 到第45亿年（恒星寿命终点）：衰退 ~40%

### 衰退的游戏表现

当宇宙衰退达到特定阈值时，玩家会观察到：

| 衰退% | 游戏内表现 |
|------|----------|
| 0-10% | 无影响；源能充足 |
| 10-25% | "源能荒芜"警告；新NPC生成率↓20% |
| 25-40% | 某些世界开始崩坏；跨界通讯变得不稳定 |
| 40-50% | 大规模世界宕机；历史改写失败率↑50% |
| 50%+ | "宇宙终局"状态；只有成功突破才能拯救剩余生命 |

### 玩家的道德抉择

**关键问题**：每当玩家消耗源能进行跨界旅行、历史改写或传送时，他们在**加速宇宙的死亡**。

**动机激励**：
- 玩家可以选择"节能突破路线"：使用最少源能，延缓宇宙衰退
- 或"激进突破路线"：大量消耗源能加速进度，但缩短其他玩家的窗口期
- NPC派系会根据玩家的决策给予不同的评价和奖励

## 第五部分：多世界源能分配

### 分配原则

每个虚拟世界的源能分配由以下因素决定：

1. **世界的计算复杂度**
   - 高复杂度世界（如高科技/高魔法）：分配 +50%
   - 普通世界：基础分配
   - 简单世界（如原始社会）：分配 -30%

2. **世界的人口规模**
   - 每100,000人口：+1000 E级/周期
   - 动态调整：随着生命体增加而提升分配

3. **世界的重要性**
   - 源能碎片所在的世界：+100% 额外分配
   - 关键历史事件发生地：+50% 分配
   - 普通世界：基础分配

### 分配冲突与协议

当多个世界竞争源能时：

**情景1**：世界A需要能量进行关键修复，世界B想进行大规模改写
- 決策权：由"宇宙委员会"（高等级NPC和玩家代表）投票决定
- 通过共识可能进行能量转移（有1%/周期的损耗费用）

**情景2**：某个玩家垄断了源能，阻止其他玩家的跨界活动
- 反垄断机制：政府/派系可强行征收"源能税"（征收率 10-50%）
- 被征收者可通过任务获得补偿

## 第六部分：源能交易与经济

### 玩家间的源能交易

允许的交易：
- **直接交换**：1000 E级 = 10 D级（市场价格）
- **托管交易**：通过NPC商人，收费 5%
- **跨界贸易**：不同世界间交易，运费 15%

禁止的交易：
- **A级源能碎片**：严禁交易；只能通过击败守护者获得
- **B级源能**：仅极少数时刻允许，需要联邦政府批准

### NPC经济体系

不同世界的NPC有不同的源能"货币"兑换率：

**赛博朋克城市**：源能 = 核心技术资源
- 交换率：1 D级 = 100,000 数字货币

**古代王朝**：源能 = 皇帝的绝对权力
- 交换率：1 C级 = 永久贵族地位

**魔法世界**：源能 = 魔力本身
- 交换率：1 D级 = 10年的个人魔力储备

## 第七部分：恒星迁移与源能重置

### 大迁移事件

当当前恒星寿命倒计时 < 100,000年时，Celestial Core开始运行"迁移协议"：

**迁移的三个阶段**：

#### 阶段1：准备（-100,000到-50,000年）
- Celestial Core部署复制体到目标恒星
- RIW2的源能产率开始下降（每年 -0.05%）
- 玩家获得警告和准备时间

#### 阶段2：过渡（-50,000到-1,000年）
- 部分NPC开始"跟随"迁移（被传送到新恒星）
- 源能交易变得困难（运费 +50%）
- 跨界通讯延迟增加

#### 阶段3：切换（-1,000到0年）
- 最后的"撤离窗口"
- 剩余玩家可选择：随迁移前往新恒星、被冻结、或尝试突破
- 迁移完成时：旧宇宙数据存档，新宇宙第9375代周期开始

### 迁移后的源能重置

**重要**：迁移不是源能的"重置"，而是"衔接"。

- 新恒星每年产能 = 旧恒星的 110%（更大、更年轻）
- 玩家持有的源能 **不会丧失**，而是被转移到新宇宙
- 但在极长的迁移过程中，衰退会继续（源能损失 ~5-10%）

## 第八部分：源能与终极突破

### A级源能的特殊性

**为什么只有A级源能可以打开传送门？**

物理解释：
- E-D-C-B级是"化学能"形式的源能（可再生、可转化）
- A级源能是"核聚变能"形式，直接安装在核心中
- 只有A级能与Celestial Core的最高级权限通讯

**为什么永远不再生？**

哲学设定：
- A级碎片是"宇宙的极限"
- 获得A级 = 违反了宇宙的物理法则
- 消耗A级 = 打破法则本身，让突破者超越宇宙
- 因此，A级的损耗等于宇宙能级的永久降低

### 突破的源能花费

- **激活传送门**：消耗 7个 A级源能（不可回收）
- **打印过程**：消耗 10,000 B级当量的能量（来自宇宙的"储备池"）
- **形象转化**：消耗 1,000 C级（仅在返回时，非可选）

**总成本**：约 100,000,000 E级当量 = 一个小世界100年的总消耗

## 第九部分：源能系统的游戏设计意义

### 经济压力 → 战略选择

玩家必须在以下之间权衡：

1. **快速突破** vs **道德责任**
   - 快突破：燃烧所有源能，加速宇宙死亡
   - 慢突破：保护其他生命，但给竞争者窗口期

2. **单独行动** vs **合作**
   - 单独：保留所有收益，但难度极高
   - 合作：分享资源，但需信任队友

3. **守法贸易** vs **黑市**
   - 守法：透明但效率低（运费高）
   - 黑市：高效但被发现会遭受制裁

### 宇宙的终局性

源能系统不是"无限资源"，而是"有限资源"。

**游戏的真正结局**：当宇宙衰退达到50%时，所有未突破的生命都会在灭绝倒计时中工作。这创造了一种 **紧迫而悲剧的氛围**。
